# Metal Slader Glory (FC) 内存/寄存器映射表

像手搓时代那样,逐字节标注用途。实机(Mesen)观测 + 反汇编(`dis6502.py` + CDL)得来。持续补充。

## RAM(零页 + $02xx-$07xx)

| 地址 | 用途 | 来源/备注 |
|---|---|---|
| `$01` | PPUMASK 影子(NMI 写 $2001) | 反汇编 NMI |
| `$17` | **自增帧计数器**,一有输入被接受就归 01(非"就绪标志") | 菜单探针 |
| `$2C` | **命令菜单光标**(0-based;子菜单也从0起);down/up 改它,A 选中 | 菜单探针 |
| `$87/$88` | **当前句子指针**(lo/hi);$F071 处执行时据此算句号 | scenetrace |
| `$9D` | 帧计时器(每帧变) | scenetrace wait_ready |
| `$B3/$B4/$B5` | NMI 保存的 A/Y/X | 反汇编 NMI($E9C3) |
| `$0200` | OAM[0].Y;**翻页箭头▽** 显示时 =`$F0`(对话等待按A) | 自动游玩 |
| `$0201` | OAM[0].tile | |
| `$0450` | **当前背景CG bank / 场景号**(0A序幕/30机甲/67通电/0B海边/10店门口/11店内/17查米近景/0E工作间/12/13/14阿源车/16穿梭机/00密语系统屏) | 全程核心 |
| `$045E` | region0(上半CG)bank;NMI 从 $0450 拷入,IRQ 应用到 $512B | 反汇编+notes |
| `$045F` | **region1(对话框)字库 bank**。原版 NMI:`=$0451`(=0);汉化改成 `=$0450\|0x80`(每场景独立字库) | probe/反汇编 |
| `$0451` | **原版对话字库 bank 号 = 0**(即 CHR bank0 = 原版假名字库) | probe 原版ROM |
| `$0454` | 立绘/头像 bank(经指针链) | notes |
| `$0461` | region1 sprite bank(IRQ 应用到 $5127) | notes |

## MMC5 寄存器($5xxx)

| 地址 | 用途 |
|---|---|
| `$5100/$5101` | PRG/CHR 模式 |
| `$5104` | ExRAM 模式(0=名字表/1=扩展属性——**可每tile选CHR bank,全量翻译的潜在钥匙**) |
| `$5113-$5117` | PRG bank($5117=$FF→末bank映射$E000;复位向量在此) |
| `$5120-$5127` | **sprite CHR bank**(8×1KB)。CONTINUE 界面写 $5127=00 |
| `$5128-$512B` | **bg CHR bank**(4×2KB)。CONTINUE 界面写 $512B=00 |
| `$5130` | CHR bank 高位(扩展 CHR>256bank 用) |
| `$5203/$5204` | 扫描线 IRQ 比较值/状态(分屏对话框靠它) |
| `$5205/$5206` | 硬件乘法器 |

## CHR 布局(汉化 build,扩到 1MB)

| bank | 内容 |
|---|---|
| `0`(CHR0=file 0x100010) | 字库基底 = 原版假名字库,**但被注入了场景67绿字的~61中文**(绿字overlay读bank0)。**CONTINUE界面也用bank0→撞这些中文→乱码** |
| `$80+场景`(如$8B海边/$94阿源车) | 汉化每场景独立字库(bank0拷贝+中文注入) |
| `$A0+场景` | **空闲**(多bank架构可用作第二字库bank) |

## 关键代码地址(CPU)

| 地址 | 用途 |
|---|---|
| `$E9C3` | NMI 入口(向量 $FFFA) |
| `$EB79` | NMI 里 `LDA $0450; STA $045E`(region0 bank) |
| `$EB7F`(file 0x7EB8F) | **字库bank设置钩子**:原`LDA$0451;STA$045F`→汉化`ORA#$80;STA$045F;NOP` |
| `$F071` | 句子处理点(据 $87/$88 算句号) |
| bank62 打字机 | `$D2ED`/`$D9EA` emit(对话正文渲染) |

## 密语/CONTINUE 机制

- **密语 = 句154(阿源车场景14)艾莉娜口述**"えりな もう おきなよ"(汉化误翻"艾莉娜 快起床啦")
- **CONTINUE = 假名输入格**(玩家选假名拼密语);用 **CHR bank 00** 渲染
- 修复要点:①CONTINUE 用的 bank00 需可读假名(但被绿字中文污染,冲突待解)②句154密语保原版假名
- **待反汇编**:CONTINUE 界面渲染代码(找它为何用bank00/用哪些tile码)、绿字overlay代码(能否让它用别的bank,腾出bank00给CONTINUE)
